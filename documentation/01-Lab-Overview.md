# Lab Overview

## Objective

The objective of this project is to design, configure, and document a simulated enterprise network using Cisco Packet Tracer while applying networking concepts aligned with the Cisco CCNA.

## Business Scenario

Robinson Technology Solutions is a fictional small business that requires a secure, scalable, and segmented network infrastructure. The network is designed to support multiple departments while providing centralized network services through a Windows Server 2022 environment.

## Project Goals

- Design an enterprise network topology
- Configure Cisco routers and switches
- Implement VLANs and inter-VLAN routing
- Configure DHCP and static routing
- Apply switch security and access control lists (ACLs)
- Practice Cisco IOS administration and troubleshooting
- Document each implementation phase using Git and GitHub
